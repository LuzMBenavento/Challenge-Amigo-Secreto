Amigo Secreto

INSTRUCCIONES

Para agregar un nombre, utiliza el botón "Añadir" o presiona enter.

Para eliminar un nombre, presiona la "X" que aparacerá cuando el cursor este sobre el mismo.

Para sortear un amigo secreto, presiona "Sortear amigo".

El amigo sorteado solo podrá ser sorteado una única vez.

El juego terminara una vez que ya no queden nombres para sortear.

Para reiniciar el sorteo, presiona "Reiniciar sorteo" y se borraran todos los nombres ingresados anteriormente.

Digite el nombre de sus amigos
